# АдминКИТ SP27 — стресс-тест и preflight

Версия: **SP27**  
Дата сборки: 2026-04-27

## Что исправлено в SP27

- Убран кастомный popover вложений, который дублировался в iOS/MAX WebView.
- Скрепка теперь открывает один системный native picker через единый input: медиатека / камера / файлы выбираются нативным слоем ОС/MAX.
- Preview медиа по-прежнему открывается мгновенно через локальный `blob:`.
- Upload комментариев идёт через `FormData/multipart` на backend и далее в MAX upload.
- Backend сохраняет лёгкий preview до внешнего MAX-upload, поэтому медиа-комментарий не должен исчезать при временной ошибке MAX.
- При ошибке MAX upload комментарий остаётся с локальным preview/fallback, а статус загрузки очищается.
- После пересылки старого поста меню раздела отправляется новым свежим сообщением вниз, а не редактирует старое меню выше.
- `dataUrl/base64` не сохраняются в `store.json`.
- Runtime/displayVersion обновлены до **SP27**.
