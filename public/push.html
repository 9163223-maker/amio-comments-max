Чтобы получать уведомления этого чата

Если АдминКИТ PUSH ещё не установлен, добавьте его на экран Домой и откройте приложение.

Если АдминКИТ PUSH уже установлен, откройте приложение с экрана Домой — мы покажем кнопку «Подключить этот чат».

Как добавить на экран Домой Я уже установил АдминКИТ PUSH Открыть инструкцию

Если приложение ещё не установлено

  1. Нажмите «Поделиться».
  2. Выберите «На экран Домой».
  3. Откройте АдминКИТ PUSH с экрана Домой.
  4. Нажмите «Включить уведомления».

Если приложение уже установлено

  1. Откройте АдминКИТ PUSH с экрана Домой.
  2. Нажмите «Подключить этот чат».
